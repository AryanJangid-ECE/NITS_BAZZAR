NITS Bazar is a smart, localized marketplace designed exclusively for the students of NIT Silchar. It features a sleek Dark Mode UI and an **AI-powered WhatsApp Bot** that automatically extracts product details from WhatsApp chats and lists them directly on the website!

---

## 🔥 Key Features
* **🤖 AI WhatsApp Integration:** Uses Selenium and Google Gemini AI to read WhatsApp messages, extract product details (Name, Price, Condition, Image), and auto-list them.
* **🛡️ Anti-Duplicate Shield:** Prevents the same item from being listed multiple times, even if the server restarts.
* **⚡ Single Page Application (SPA):** Lightning-fast navigation between Marketplace, Sell Item, and Dashboard without page reloads.
* **🎨 Hacker/Gamer UI:** Premium Black & Red color scheme with hover animations and responsive grid design.
* **📱 Direct WhatsApp Routing:** "Contact Seller" button automatically opens a WhatsApp chat with the seller's number pre-filled.

---

## 💻 Tech Stack
* **Frontend:** HTML5, CSS3 (CSS Variables, Flexbox/Grid), Vanilla JavaScript
* **Backend:** Python, Flask
* **Database:** JSON (Local Storage for MVP)
* **Automation & AI:** Selenium WebDriver, Google Generative AI (Gemini Flash)
